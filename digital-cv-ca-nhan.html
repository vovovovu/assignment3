Goals

Goal 1

Áp dụng các kiến thức đã học về HTML cũng như CSS để xây dựng một trang Digital CV

Goal 2

Trang web có nội dung phù hợp với bản thân theo cấu trúc như bản thiết kế yêu cầu.

Goal 3

Tạo nên một trang CV cho chính bản thân mình, mang phong cách và cá tính riêng của chính mình dựa trên cấu trúc cho sẵn.

Scopes

IN

Tài nguyên có sẵn là một template project với cấu trúc như sau:

  Assignment 01 Stater
 ├─ css
 │ └─ styles.css
 ├─ images
 │ └─ default-avatar.png
 │─ index.html
 └─ Assignment Text.pdf

  • index.html: File HTML chứa code để tạo cấu trúc cho trang Web.
  • styles.css: File CSS chứa code để tạo kiểu cho trang Web.
  • images: Folder chứa các hình ảnh sẽ sử dụng trong bài Assignment.
  • Assignment Text.pdf: File PDF bao gồm thiết kế của trang Web và các đoạn Text mà bạn sẽ sử dụng.
OUT

Dựa vào template project hoàn thành project theo đúng cấu trúc đã cho, xây dựng trang web đúng theo bản thiết kế.

File zip project bao gồm các file cần thiết trong project.
